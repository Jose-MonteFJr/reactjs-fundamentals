# Fundamentos do React 

## Guia

Siga o guia nessa ordem de aprendizado e revisão

- **structuring/** -> Pasta com arquivo `README.md` para explicação inicial do React, estrutura e conceitos base.

- **properties/** -> Pasta com arquivo `README.md` para explicação de props (propriedades) e arquivo `properties.html` para testar os conceitos.

- **components/** -> Pasta com arquivo `README.md` para explicação de componentes e arquivo `components.html` para testar os conceitos.

- **events/** -> Pasta com arquivo `README.md` para explicação de eventos no React e arquivo `events.html` para testar os conceitos.

- **hooks/** -> Pasta com arquivo `README.md` para explicação de hooks e arquivos para testar os conceitos.

- **calculator/** -> Projeto para colocar em prática todos os conceitos aprendidos.

- **model.html** -> Modelo HTML utilizado como base para arquivos testes.
